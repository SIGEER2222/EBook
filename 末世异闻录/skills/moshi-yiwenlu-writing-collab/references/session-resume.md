# 断点续接模板

这份模板用来收尾和交接。哪怕只写三分钟，也尽量留一版。

## 一、最小续接记录

```md
当前任务：
已完成：
卡住点：
下一步：
必读文件：
未收项：
```

## 二、推荐补充

- 当前章节号或当前设定文件名
- 这次已经确认过的口径
- 这次故意没动的东西
- 下次一上来最该先做哪一步

## 三、适合本项目的写法

```md
当前任务：修第 12 章章尾，把新日教的正面秩序感立住，同时埋一丝收容权的问题
已完成：重写了中段冲突，补了路权和配给细节
卡住点：章尾回声还偏弱，林砚的判断有点太直给
下一步：重写最后 600-800 字，让代价先落到普通人身上，再让林砚吐槽一句收口
必读文件：
- 设定圣经/剧情设定/2026-04-04《末世异闻录》作者写作备忘单.md
- 背景完善/宗教格局与神权结构.md
```
