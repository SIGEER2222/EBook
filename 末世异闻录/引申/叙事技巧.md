# 1. 阻碍困境
阻碍，作用是叠加情绪，给剧情添加逻辑、真实性，重在在于**剧情需要**。在实际写作中，有可能因为突然有了更巧妙的解决方法灵感，或代入人物时，为了真实细节，从而改变设定阻碍的方式，所以不强行设定不可改变的阻碍前提，一切服务于表达该阶段的情节目的。

1. **人为暴力阻碍**：敌方用暴力、权力手段，强制让主角陷入困境，如：追杀、围堵、上级警告。
2. **人为谋略阻碍**：敌方运用计谋设置陷阱，给主角添加阻碍，陷入困境，如：栽赃、收买背叛。
3. **规则限制阻碍**：敌方利用权力或超凡力量，添加规则限制，如：强纳税、倒计时、直视催眠。
4. **阶层规则阻碍**：上层、将军、队长等人物对组织内人员的命令，如：见人就杀、栽赃某人。
5. **社会规则阻碍**：当前场景中的社会背景规则阻碍，如：习俗、末日、资源垄断、种族歧视。
6. **自然场景阻碍**：合理制造天气地形自然类阻碍，最好作为伏笔，如：雷雨、沙漠、瘟疫。
7. **前提不足阻碍**：拥有解决问题的方法，但准备的还不够充分，如：实力、信息、证据不足。
8. **自身内心阻碍**：心理阴影、内心恐惧、受伤未痊愈、自我阻碍，如：面对颤抖、旧伤复发。

# 2. 结局反馈
是对整个情节单元的收尾和盘点，对故事主题的彻底展露与回味，满足反馈读者情绪，进行下一个故事的铺垫。  
**满足读者情绪**，给予读者在冲突过程中积累的期待一个交代，形成爽点或满足感。  
**展示成长与变化**，清点主角在这次事件中的收获与损失，体现其成长。  
**承上启下**，为下一个情绪事件和欲望目标埋下伏笔。

具体表现：

① **直接收益**：对欲望目标的反馈，物质反馈，资源获取，地位提升，关系确立，目标达成，弥补付出；  
② **精神反馈**：反击嘲讽，打破质疑，证明自己，自我实现，成长深思。  
③ **人物反应**：对手和敌方的不可置信，朋友和爱慕者更多的崇拜，路人旁观者表现震惊和敬畏，大人物和舆论的关注。  
④ **伏笔延续**：情节结束后，遗留下的线索和未知，对下一个情节进行铺垫和延续。情绪体验与阅读满足感。

# 3. 情绪事件
矛盾，在小说中通俗来讲就是在一个场面下不可调和对立的状态。矛盾是一种抽象的概念，是文学用来分析已完成作品的视角，不可拿来当作写作思维，但是可以做情绪事件起因的源头：

1. **情感关系类矛盾**：付出与回报的不平等、血缘家族规则的束缚、帮助友方摆脱困境。
2. **资源利益类矛盾**：金钱、宝物、知识、异性、权利等所有人都渴望的东西，进行争夺。
3. **权利地位类矛盾**：权利斗争、职场斗争、上下级关系、以权谋利、争夺领导权。
4. **生存竞争类矛盾**：情敌对抗、竞争比赛、大逃杀、末日，在某个规则下的争斗。
5. **虚荣恩怨类矛盾**：误会蒙羞、恩怨情仇、虚荣面子、嫉妒使人面目全非，现实更魔幻。
6. **阵营观念类矛盾**：正义邪恶、三观不合、阵营对抗，自古正邪势不两立，谁对谁错？
7. **阶级社会类矛盾**：身份、种族、阶层、血统，天不知人间疾苦，人不知天上宫阙。
8. **现实道德类矛盾**：信任、公平、守信、责任、文明等，人物与现实三观的矛盾。
9. **人物自身类矛盾**：自我与社会、理想与现实、道德和利益，人物自身面临两难选择的矛盾。

# 4. 整体结构
## 宏观视角
**故事目的**：是为了**确定题材主题**，根据主题将灵感片段扩充成一个**分阶段大故事**，再设定基本世界、情节和人物，确定这个故事用谁、在哪、该怎么讲。

### 三，细纲阶段（情节目的层面）
**材料**

+ 场景设定：根据剧情需要，细化当前场景
+ 人物设定：根据剧情需要，功能类型创建

**形式**

+ 基本形式：目标开始、期待发展、精彩冲突
+ 基本结构：目标、困境、方法、冲突、解决

**动力**

+ 读者欲望：欲望推动、构建期待、情绪动态
+ 角色欲望：矛盾建立、持续对抗、最终结果

## 微观视角
事件是具体写作层面，除了根据剧情需要细化故事、人物、目的、场景外，目的是通过**写作时控制信息的展露、制造人物的信息差距**，来制造反转、反差、戏剧冲突的效果，以控制读者情绪。

## 二，构思阶段
+ 真实世界经验
+ 虚构世界经验
+ 个人抽象灵感

## 大概要写什么故事？
### 二，大纲阶段（故事目的层面）
**材料**

+ 世界设定：资源、地图、阶层、势力等
+ 人物设定：立体人物、扁平人物、路人等

**形式**

+ 阶段分化：阶段1、阶段2、阶段3……
+ 整体结构：总分总式、阶梯式、并列式等

**动力**

+ 读者目的：情欲、个欲、求知欲等九大欲望
+ 角色目的：权色名利、虚荣信仰、贪嗔痴食

## 中观视角
**情节目的**：因为阅读完整，才能感受故事主题，所以情节的目的，就是通过**构建场景、人物、情节**，来**引起读者的起始阅读欲望、和持续阅读欲望、以及结果的情绪反馈**，一切为了让读者阅读下去的主要目的。

### 四，章节阶段（事件目的层面）
**材料**

+ 信息视角：旁白、全知、人物、敌方、友方
+ 人物细化：身份、性格、言语、喜好、目的

**形式**

+ 信息输出：控制读者、主角、他人的信息差
+ 视角结构：选择视角让故事符合起承转合

**情绪**

+ 读者情绪：上行、下行、持续动态、落差
+ 角色情绪：引发原因、激励情绪、情绪反馈

# 5. 基本结构
**遭遇事件 → 欲望目标 → 阻碍困境 → 解决方法 → 行动解决 → 结局反馈**

示例：

**主角1级**，父母双亡，**3级人物欺负主角最重要的人**，爷爷或妹妹，**想要变强**，金手指觉醒，隐藏成长至3级，主角反击，打败3级人物，事件解决。

**主角3级**，与**5级人物竞争排名/宝物/任务**，打败5级人物时，暴露某样珍贵物品和信息，得知**8级人物想要迫害自己**，隐藏成长为7级并将功法修炼至大成，打败8级人物，解决危机，事件解决。

**主角10级**，一次任务中途**得罪大家族12级子弟**，打败12级人物，找来**15级哥哥**，在绝境中重伤15级家族天才，**18级大长老要杀了主角**，主角躲避误入秘境中获得传承突破，反杀，事件解决。

总结：  
小白升级文，剧情非常简单，只需要懂得通过各种欲望开启故事，能力提升后，反击即可。玄幻修仙、奇幻武侠、废土末日、游戏异界、新颖的宏大世界、奇异的技能功法、多样的装备武器和刺激的战斗场景，足以让爱好此类的人乐此不疲。当然有更好的故事剧情就完美了。此类文很多，除了世界设定外，能够脱颖而出的关键所在，是对情绪的把握、逻辑性、阻碍和解决方法的惊艳度。

# 7. 行动解决（转折高潮）
图中曲线节点：

+ 情绪事件
+ 阻碍困境
+ 解决方法
+ 反转转折
+ 解决获得
+ 欲望目标

坐标含义：

+ 纵轴：情绪线
+ 横轴：事件线
+ 上方情绪：爽
+ 下方情绪：伤心

右侧说明：

① **高潮的本质**：是积累叠加爆发，是从量变到质变的节点，是凸显主题最为核心的地方，高潮不是凭空出现的，而是前面所有铺垫、矛盾、情感积累的必然结果。

② **高潮的形式**：铺垫决定爆发力，高潮是否精彩，大部分取决于前面情节的铺垫，读者要经历前面主角的挣扎、付出、希望与失望，才能在最后的爆发时产生强烈的情绪共鸣。

③ **高潮的目的**：避免平庸，表现与深化主题，通过极端、冲突、强烈的选择和后果，来调动读者最强烈的情绪波动，提供超越日常的情感体验和满足感。

# 8. 欲望目标
1. **生理需求-生存欲望**：遭受迫害、生命危险时，会尽一切保护自己的生命。
2. **安全需求-需求贪欲**：充足的食物、更多的钱、消除危机，人类的贪欲是永无止境的。
3. **爱与归属-情感欲望**：对异性的渴望，对亲人朋友的情感，对共同利益者的情感维持。
4. **尊重需求-个人价值表现欲**：别人没有我有的虚荣感，别人不会的问题我会、别人不行我行。
5. **自我实现-社会价值表现欲**：渴望真理，渴望社会认同，渴望在集体中可以证明自己。
6. **外在工具-工具欲**：遇到问题时，渴望得到帮助自己完成目的的工具，武器装备，甚至是人。
7. **超凡力量-神器欲**：高级工具欲，渴望超凡的力量，创造工具、学会特长、自身天赋。
8. **未知渴望-求知欲**：认知属于身体被动欲望，求知欲望强烈程度取决于自身所面临的事情。
9. **符合预期-掌控欲**：高级求知欲，人类会根据已知信息推演将要发生的事情，渴望符合预期。

# 9. 主题目的（可叠加）
## 1，情绪体验的故事
写孤独、恐怖、热血、刺激、浪漫、暖味、治愈、荒诞等，重体验和情绪的故事，构建以情绪感受为核心的事件情节，让读者可以感同身受，从而感受到这些情绪。

## 2，人物情感的故事
写爱情、友情、励志、选择、人性、英雄、群像等，重人物和情感的故事，构建以塑造人物为核心的事件情节，让读者代入、共鸣，对人物感兴趣，就会自然而然地产生期待。

## 3，世界设定的故事
写规则、秩序、灵异、魔法、异能、科幻、末日等重世界观和设定的故事，构建以新颖魔幻的世界为核心的事件情节，让读者进入另一个由想象力创造的神奇世界，让读者感受到不同的规则。

# 10. 视角选择
以下是为了理解视角转换目的，和视角转换技巧和转换逻辑，需要明白的是，没有最好，只有合适，高手一般根据剧情需要，在合适的时机转换视角，而读者不会察觉出，可以选择的视角有：

1. **旁白镜头视角信息**：不进行深度解析和人物心理，不过多解释场面和人物行为的旁白信息视角，只叙述旁观者可以看到的，类似于镜头视角。
2. **上帝全知视角信息**：与镜头视角相反，该视角解释所有人的动机、行为等，信息全知，只在解释剧情的时候使用，让读者可以更加轻易地理解信息，降低阅读理解难度，但一定要适度使用。

只用一个人的视角去叙述信息，除了该人物需要知道的，其它信息全都不知，这便是人物个人主观视角。为了好理解和操作使用，我将人物主观分为主角、友方、敌方、旁观限制级主观信息视角，以达到写作时的思路选择。

3. **主角主观视角信息**：除了该人物需要知道和已经知道的，其它信息全都不知。
4. **敌方主观视角信息**：运用视角转换，将主视角转换到敌方视角叙述，以达到表现主角的目的。
5. **友方主观视角信息**：同样，将主视角转换到友方视角叙述，看待主角的行为。
6. **旁观主观视角信息**：背景人物旁观视角叙述，一般不作为主视角，作为补充信息和叠加情绪的作用，是一些群众的视角和声音。
