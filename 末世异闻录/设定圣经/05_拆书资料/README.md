# 拆书目录说明

> 用途：存放每次“拆书环节”的本地输出文件。

---

## 规则

以后每次拆一本书，默认都要：

1. 在本目录下新增一份单独文档；
2. 文件名格式优先使用：`书名拆解.md`；
3. 文档内容默认包含：
   - 一句话定位
   - 核心看点
   - 核心爽点
   - 结构引擎
   - 人物功能位
   - 节奏与情绪
   - 对《末世异闻录》的可借用点
   - 不可照搬点
   - 转译建议
