# 人类与虫群的相处模式

> 目标：补足“人类为什么能在虫群长期压迫下继续活着”这一层设定。  
> 用途：统一边地日常、清巢制度、路线治理、虫材经济、民俗禁忌和政治争执的写法。  
> 核心结论：所谓“虫群”，并不只是成片涌动的虫潮，而是区域首脑统筹下的多支异种群落。主宰给人类留出了基本生存区，于是人类与虫群的长期关系，更接近边界摩擦、外沿争夺和局部清剿。

---

## 一、总定义

人类与虫群的关系，长期都处在一种高压共处状态。

主宰长期实行圈养政策，给人类划出了若干可以稳定繁衍和经营的生存区。  
也正因为这样，人类和虫群并不是天天全面开战。真正反复出事的地方，主要集中在这些生存区之外的拓展地、采掘带、旧路和新垦区。

所以“虫群”这个词，在人类口中更像一个统称。  
它并非单指某一种虫类，而是一整个大区里受同一首脑约束的多支异种群落。  
其中既有典型虫类，也有更接近虫兽、异化掠食者、寄生种和地下掘进种的成员。

大多数年代里，人类做不到把虫群彻底从地图上抹掉；  
虫群也不会把每一寸土地都时时刻刻压满。  
真正意义上的大规模虫潮十分少见，更多时候出现的是零散越界、小群试探和局部聚集。

于是人类真正掌握的能力，逐渐变成了八个动作：

- 预警；
- 测绘；
- 切割；
- 诱导；
- 封堵；
- 回收；
- 撤收；
- 复占。

这八个动作，才是人类面对虫群时最稳定的文明本能。

所以边地治理的重点，永远落在三件事上：

- 提前知道虫群从何处来；
- 尽量让它们沿着对人类伤害更小的方向走；
- 在付得起代价的前提下，把关键路线和关键节点保下来。

## 二、人类对虫群的日常认知

在这个世界里，普通人从小就会被教会几条最基础的虫群常识。

### 1. 土地要按风险来认，不按归属来认

一片地在账面上归入何城名下，很多时候意义不大。  
更重要的是它现在处在下面什么状态：

- `净线地`：巡逻和哨站能稳定覆盖，适合耕种和走商；
- `扰动地`：近期有探虫、标示虫或地表鼓动痕迹，必须限时通行；
- `回灌地`：虫群会周期性返潮，适合设伏、设饵和短期开采，不适合久留；
- `弃置地`：已失去维护价值，只保监视，不保常驻。

边地居民看地图，先看风险层，再看行政线。

### 2. 孤虫常常比成群目标更危险

真正让人心里发毛的，未必是远处黑压压的一片。  
一只出现在不该出现位置上的探虫，往往意味着后面已经有更完整的路线试探。

所以边地有句老话：

**看见落单的虫，先想它背后那条路。**

### 3. 地下永远要留一分戒心

地下空间在虫群战争里没有天然安全感。

老井、废窖、矿洞、涵道、地基裂缝、潮湿菜窖，都可能变成孵化点的外延。  
城里但凡听见地底空鼓、夜里持续细震、牲口无故躁动，值夜人就必须上报。

### 4. 边地常识比神话更重要

一个孩子在边地长大，先学会的通常不是史书，  
而是下面这些东西：

- 巢壳碎片长什么样；
- 土层拱起和鼠洞翻土有什么区别；
- 看见什么分泌痕和蜕壳必须退；
- 夜里听到什么频率的震颤要关灯闭门；
- 驮兽为何会突然拒绝过桥；
- 什么尸体不能碰，什么残骸要立刻焚掉。

虫群压力把很多生存知识压成了接近本能的经验。

## 三、节点社会如何围着虫群运转

人类的节点文明，表面上围着城、防线、商路和工坊转，  
深一层看，其实也围着虫群的活动节律转。

### 1. 巡逻的首要目的，是争时间

边地巡逻队最重要的职责，不是见虫就杀，  
而是尽快确认四件事：

- 这一波是探路、补巢，还是准备聚群；
- 它们试探的目标路线是什么；
- 附近是否已有新的孵化点；
- 这条线还值不值得继续保。

很多巡逻记录的价值，比一次小规模胜负更大。

### 2. 清巢季是制度，不是临时行动

只要地方财政还撑得住，很多城邦都会固定举行`清巢季`。

常见时间点包括：

- 冻土初解之后；
- 秋收之前；
- 重要商路重开之前；
- 城外新耕带启用之前。

清巢季的目标通常很明确：

- 烧掉外沿卵堆；
- 清理废屋、旧井和浅穴；
- 重画安全半径；
- 回收还能用的虫材；
- 把“能住一季”和“不能住一季”的地方重新分出来。

它像农时，也像兵役，更像每年都绕不开的公共工程。

### 3. 只保路线，不保整片地

这是人类与虫群长期周旋后形成的核心经验。

人类不会为了面子把整块荒地都纳进控制半径。  
更现实的做法是把资源压在少数关键对象上：

- 粮路；
- 水路；
- 护矿道；
- 城间军路；
- 近域耕带；
- 港外补给岛。

一片地只要还能给路线充当缓冲区，它就还有价值。  
一旦维护成本高过路线收益，地方执政团就会考虑主动弃守。

### 4. 弃线与复占都已经制度化

在人类文明里，弃线不再自动等同于耻辱。  
边地真正丢人的事，通常是：

- 明知守不住还拖着不撤；
- 明知已成虫巢外延还逼平民复耕；
- 为了账面领地拖垮整条后方路线。

相反，有经验的官员和指挥官会把弃线、复占、再测绘做成一整套连续动作。  
在很多地方，一条路这一代人能修三次，也能弃三次。

## 四、争夺区的人类如何和虫群周旋

争夺区的生存逻辑最能体现“相处模式”这四个字。

这里最常见的冲突，不是铺天盖地的大潮，  
而是某条线外突然多了几十只、某片矿带冒出一窝、某支异种开始越过旧界。

### 1. 诱导比硬顶更常见

只要还有选择，人类往往更愿意改虫群的方向，而不是和它们硬碰硬地耗。

常见办法包括：

- 在废弃带放热源假标；
- 用血饵和腥料把小股虫群引向空地；
- 提前炸塌旧涵道和废矿井；
- 故意留出一条看似更容易走的死路；
- 在不保的村落外围留假仓、假棚和假畜栏。

边地的聪明，很多时候都花在“把灾祸往别处偏半步”上。

### 2. 设饵和断饵同样重要

有些地方会在争夺区外缘设置固定诱饵点。  
诱饵可以是腐肉、牲口血、热灯、震源器，也可以是经过处理的虫尸。

但诱饵点不能无限长期存在。  
一旦虫群开始把诱饵点当成稳定坐标，人类就要立刻断饵、迁点、清痕。

诱饵本身也是一门很贵的手艺。  
用得好，能给一条路换来半个月平稳期；  
用不好，等于替虫群校准路线。

### 3. 孵化点很少一次清干净

前线清巢最常见的情况，是把一个孵化点打残、封死、烧空，再盯它后续会不会回长。  
真正意义上的彻底根除，代价常常大到不适合写成日常胜利。

于是边地会出现很多“半死巢”“封存巢”“盯防巢”的说法。  
这些地方不是安全地，只是暂时被压住了。

### 4. 采掘、耕种、放牧都会跟着虫群节律走

争夺区的生产活动很少常年稳定。

更常见的是：

- 春季清一圈，抢种一季短粮；
- 夏季保住水车和路口，缩小耕带；
- 秋季在外沿聚群扩大前抢收；
- 冬季回缩人口，封掉外围棚屋和牲口栏。

边地很多村子并不追求全年常住，  
而是把自己活成一群会迁移的季节性附属节点。

## 五、经济层面的利用与依赖

虫群带来死亡，也带来了非常现实的材料经济。

### 1. 虫材回收是正式产业

很多地方都有合法备案的虫材回收队。  
他们处理的主要东西包括：

- 甲壳；
- 毒囊；
- 感器；
- 节肢骨材；
- 韧膜；
- 腺液；
- 虫蜡；
- 卵壳。

这些材料会流向：

- 护甲工坊；
- 药剂坊；
- 炼金坊；
- 灯油和封蜡作坊；
- 陷阱和警报器具工坊。

边地有不少城镇，外表像普通堡镇，账本上却有一大半收入靠虫材吃饭。

### 2. 虫巢采掘是高风险高利润行业

废弃虫巢本身也有开采价值。

能从巢里挖出来的，往往包括：

- 未孵化卵层；
- 可提炼的腺块；
- 稀有壳层；
- 被虫群搬进去又没来得及回收的金属和尸骨；
- 围巢伴生的特殊菌类与药草。

这类生意利润极高，死亡率也高。  
于是很多地方会把它做成半军管、半承包的模式。

### 3. 风险承包长期存在

有些脏活累活，城邦不愿自己直接做，  
就会放给备案队伍或行会承包。

常见项目包括：

- 清理浅层孵化点；
- 回收大潮后的残骸；
- 修复被虫群掏空的路基；
- 进入弃镇带出账册、机件和标准件；
- 给争夺区重新钉界桩。

这类承包能让边地始终维持一种“危险也能折成价格”的现实气质。

### 4. 黑市会把风险吃到更深处

只要合法市场有门槛，黑市就会伸进来。

黑市最常交易的东西包括：

- 未登记的高危虫材；
- 还能再孵化的卵层；
- 寄生体样本；
- 禁用诱饵配方；
- 伪造的清巢凭证；
- 被判定必须焚毁却被偷偷留下的异化器官。

因此虫群既是外患，也会不断反过来腐蚀人类自己的制度。

## 六、文化与日常习俗

一个和虫群打了很多代交道的文明，民俗一定会留下非常深的痕。

### 1. 童谣、俚语和报时方式都带虫味

边地童谣里常有“听地”“数震”“关灯”“离井口远些”的内容。  
很多地方连报时都和虫群活动有关。

例如：

- 第一轮城钟响，提醒外棚收火；
- 第二轮城钟响，提醒牲口归栏；
- 夜巡梆声变急，表示外墙外发现地表鼓动；
- 连续短钟，表示马上闭门。

### 2. 成年礼常常带着清巢意味

在和平农耕世界，成年礼可以更柔软些。  
放到这里，很多地方的成年礼会要求年轻人完成下面几项之一：

- 认巢纹；
- 走夜哨；
- 记一段安全路图；
- 学会封井和点驱虫烟；
- 跟着回收队抬一次虫甲。

这会让一个人的成年，不只是身体长大，还意味着正式进入防线生活。

### 3. 房屋和仓储都有针对虫群的小设计

虫群压力会把很多细节磨进建筑里。

例如：

- 粮仓底层抬高；
- 门槛包铁；
- 屋后预留焚坑；
- 水井加双层盖；
- 仓房四角埋警铃线；
- 牲口栏与居住区留出隔离带。

普通百姓未必懂大型术式，  
但会很懂怎么把屋子改得更不容易被掏空。

### 4. 勇敢的定义会被改写

在这种文明里，受尊敬的常常不是冲出去杀得最猛的人。  
真正能让人服气的，多半是：

- 提前发现巢线的人；
- 把一群孩子带回城的人；
- 在断路前最后一次把药送进来的队伍；
- 愿意把虚报安全半径的官吏揪出来的人。

这种价值观会让整个人类社会更看重可靠、耐性、记性和责任。

## 七、政治和治理上的长期拉扯

虫群长期存在，意味着政治争执也会长期围着虫群转。

### 0. 冲突规模有一套很现实的处理顺序

边地不会动不动就吹响全面战争号角。  
更常见的应对顺序有三层：

- 单只、零散袭击、短距潜伏目标，通常交给猎魔人、承包清剿队、教会武装或地方外勤处理；
- 数量上来之后，地方守备队和城防法师会接手，优先保村镇、保路口、保井口；
- 一旦某支异种群开始持续越界，或者数量已经威胁到节点和耕带，才会上报领主，征调军队和正式法师队伍。

这套层级能让边地财政不至于被频繁拖死，也解释了猎魔人和民间武装为什么始终有活路。

### 1. 围绕虫群而生的岗位会很多

一个成熟城邦，往往会专门设立下面这些职能：

- 测绘署；
- 清巢队；
- 地下封检司；
- 路权署；
- 回收备案所；
- 争夺区承包审验处；
- 边民迁收官；
- 孵化点档案库。

这些岗位平时看着不显山不露水，真出事时却比礼官和税吏重要得多。

### 2. 地方和中央总会为“清到何处、退到何处”吵个不停

中央看的是整条粮线、军线和总账。  
地方看的是本地村庄、矿带、税源和人心。

于是冲突会反复出现：

- 母城要求撤掉外围村镇，地方豪强舍不得税和地；
- 战区要求断路止损，商路同盟还想再搏一次通行窗口；
- 学派要求封存危险巢穴，承包行会却想进去发财；
- 执政团宣称安全半径已恢复，测绘署的图纸却完全对不上。

这些争执能自然长成很强的剧情推动力。

### 3. 领地观会被虫群压力逼得非常现实

账册上的边界，很多时候只能决定税收归属。  
真正能决定统治力的，还是：

- 还能过的路线；
- 还能用的井口；
- 还能种一季的地块；
- 还有人值守的哨塔。

因此很多政权嘴上说争地，  
实际争的是城链、桥头、港口、矿道和中继塔。

### 4. 新秩序最容易靠“保路和清巢”赢得人心

旧统治一旦在这些事情上无能、迟钝、贪腐，人心就会迅速滑走。  
一个新势力只要真能做到下面几件事，就会立刻显得很像救世者：

- 不虚报虫讯；
- 不拿清巢军费做账；
- 不把边民扔在错误的安全半径里；
- 让撤离和复耕都有章法；
- 让回收队和测绘队的死亡不再白费。

虫群压力会放大治理能力的差距，也会放大统治者的丑态。

## 八、对剧情最有用的几种虫群型冲突

这套设定落到正文里，最容易长出下面几类戏。

### 1. 路线戏

一条路还能不能保，常常比打死多少异种更重要。  
护送、抢修、封路、误判虫讯、真假安全窗口，都能直接撑起一章到一卷。

### 2. 村镇戏

村民想留下收最后一轮粮，  
执政团想提前撤，  
回收队想趁乱发财，  
测绘署却认定地下已经空了。  
这种局面一摆出来，人物和立场会自动活起来。

### 3. 官僚戏

图纸、账册、虫讯等级、清巢经费、死亡名册，只要有一项被做手脚，  
一场“明明能救却没救下来”的惨案就成立了。

### 4. 冒险戏

进入封存巢、废弃镇、旧矿井、回灌带，去抢一个人、一份图纸、一台核心部件，  
这类行动既有风险，也天然带着世界异闻感。

### 5. 文明戏

真正高级的戏，不只停在杀虫。  
更有力量的部分，往往在于角色要决定：

- 该保的线；
- 该放弃什么；
- 先撤的人口；
- 必须翻出来的账；
- 这块地方值不值得用一代人的命去换。

## 九、一句话定稿

《末世异闻录》里的人类，与虫群的关系最好写成一种被圈定边界后的长期高压共处。大规模虫潮极少，常态是外沿摩擦、局部聚群和边界试探；人类则靠猎魔人、守备队、领主军和法师团层层接手，把少数路线、节点和拓展窗口一点点保下来。
